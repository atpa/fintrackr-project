# FinTrackr Project Roadmap

This document outlines the future direction and potential features for the FinTrackr application.

## Q1 2026: Core Feature Enhancement

-   **[ ] Budgets Module:**
    -   Allow users to create monthly budgets for different categories.
    -   Track spending against budgets in real-time.
    -   Visualize budget progress on the dashboard and reports page.

-   **[ ] Financial Goals:**
    -   Users can set savings goals (e.g., "Vacation Fund", "New Car").
    -   Track progress towards goals by linking them to specific savings accounts.

-   **[ ] Transaction Tagging & Rules:**
    -   Implement a tagging system for more granular transaction analysis.
    -   Create rules to automatically categorize or tag transactions from specific merchants.

## Q2 2026: Advanced Analytics & User Experience

-   **[ ] Advanced Reporting:**
    -   Net worth tracking over time.
    -   Cash flow analysis (monthly/yearly).
    -   Customizable report generation (e.g., by date range, accounts, categories).

-   **[ ] User Profile & Settings:**
    -   Allow users to manage their profile information.
    -   Add customizable currency and date format settings.
    -   Implement theme switching (Light/Dark mode).

-   **[ ] Investment Tracking:**
    -   Basic module to track stocks, bonds, and cryptocurrency holdings.
    -   Manual entry of assets and their current value.

## Q3 2026: Connectivity & Mobile

-   **[ ] Plaid Integration (Proof of Concept):**
    -   Research and implement a test integration with Plaid for automatic bank account synchronization.
    -   Focus on read-only access for transaction and balance fetching.

-   **[ ] Progressive Web App (PWA):**
    -   Make the application installable on mobile and desktop devices.
    -   Enable offline access for viewing cached data.

-   **[ ] Push Notifications:**
    -   Alerts for large transactions, upcoming bills, or budget overages.

## Long-Term Vision

-   **[ ] Multi-user / Family Sharing:** Allow users to share accounts and budgets with family members.
-   **[ ] Machine Learning for Financial Insights:** Proactively identify spending trends, suggest budget optimizations, and forecast future balances.
-   **[ ] Bill & Subscription Management:** Automatically identify recurring payments and remind users of upcoming due dates.
